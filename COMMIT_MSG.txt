feat(ui): 투자 정보 화면 레이아웃 개선 및 InputWithUnit size 변형 추가

투자 정보 수정/상세 화면의 레이아웃을 모드별로 최적화하고, InputWithUnit 컴포넌트에
size 변형을 추가하여 일관된 UI를 제공한다.

[주요 변경 사항]

1. InputWithUnit 컴포넌트 개선
   - size prop 추가: 'sm' | 'md' | 'lg' 변형 지원
   - 기본 너비를 w-40(160px)에서 w-full로 변경하여 컨테이너 전체 너비 사용
   - size별 패딩·텍스트 크기 정의 (sm: py-1.5 text-sm, md: py-2 text-sm, lg: py-3 text-base)

2. 투자 정보 수정 화면 레이아웃 변경
   - 수정 모드: 라벨과 인풋을 상하 배치 (space-y-1.5)
     - 라벨: text-coolgray-900 font-bold text-sm (위)
     - 인풋: InputWithUnit (아래, w-full)
   - 표시 모드: 라벨과 값을 좌우 배치 (flex justify-between)
     - 라벨: text-coolgray-500 (좌측)
     - 값: font-semibold text-coolgray-900 (우측)
   - 월 투자금, 목표 기간, 연 수익률, 매월 투자일 모두 동일한 패턴 적용

3. 연 수익률 제안 칩 개선
   - InvestmentEditSheet 칩 크기 축소: text-sm → text-xs, px-3 py-1.5 → px-2 py-1
   - 인풋 필드 기준 우측 정렬 (flex justify-end w-full)

[파일 변경]
- components/ui/input-with-unit.tsx: size 변형 추가, w-full 기본값 적용
- app/components/InvestmentDetailView.tsx: 모드별 레이아웃 분기 처리
- app/components/InvestmentEditSheet.tsx: 칩 크기 축소

[결과]
- 수정 화면에서 라벨과 인풋의 상하 배치로 가독성 향상
- 상세 화면에서 기존 좌우 배치 유지로 일관성 확보
- InputWithUnit size 변형으로 다양한 화면에서 재사용성 향상
